# Data Sources

## Primary Dataset
**Our World in Data — COVID-19 Dataset**
- URL: https://ourworldindata.org/covid-deaths
- Format: CSV
- Coverage: 63 countries, 2020-2026
- Update frequency: Daily
- License: CC BY 4.0 (free to use)

## Fields Used

### Cases & Deaths
| Field | Description |
|-------|-------------|
| new_cases | Daily new confirmed cases |
| new_deaths | Daily new confirmed deaths |
| total_cases | Cumulative cases |
| total_deaths | Cumulative deaths |
| case_fatality_rate | Deaths / Cases * 100 |

### Vaccinations
| Field | Description |
|-------|-------------|
| people_vaccinated | Total people with at least 1 dose |
| people_fully_vaccinated | Total people fully vaccinated |
| people_vaccinated_per_hundred | Vaccination rate % |

### Country Metadata
| Field | Description |
|-------|-------------|
| country | Country name |
| code | ISO country code |
| continent | Geographic region |
| population | Total population |
